Matplotlib Exercises¶

  • The goal of the exercises
    • Get comfortable with Matplotlib

Step 1¶

  • Then import matplotlib.pyplot as plt

# Plot 1:
fig, axs = plt.subplots(2, 2, figsize=(10.5, 8))

  • Make a new figure
  • Make a bar chart with volume for January 2020

fig = plt.figure(figsize = (10, 5))
 
read_csv.index.values

 # Filter dates with January month
january_dates = read_csv[(read_csv.index.month == 1 ) & (read_csv.index.year == 2020 )]

# Append '00:00:00' to the timestamp of the filtered dates
january_dates.index = january_dates.index.strftime('%Y-%m-%d') + ' 00:00:00'

january_dates = january_dates.sort_index()

plt.bar(january_dates.index , january_dates['Volume'], color ='steelblue',
        width = 0.5)

plt.xlabel("Date")
plt.xticks(rotation=90)
plt.show()
